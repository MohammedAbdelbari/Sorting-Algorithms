Sorts the range [first, last[ using bogo sort algorithm.

Runs in O((n + 1)!) in the average case and has an unbounded runtime in the worst case. Has a space complexity of O(1). This function should only be used for educational purposes.

Parameters
firstAn iterator pointing to the first element in the range.
lastAn iterator pointing to the element that comes after the last element in the range.
comparatorThe comparator boolean function used to compare each two elements while sorting.
